# SCTI 2025 - UENF
Material para palestra e minicurso na Semana Acadêmica de Computação da UENF 2025.

## Agenda

**04/09/2025 (quinta-feira) - 17:00-18:30**
- **Palestra**: Erlang, MCP e Kubernetes: Lições de um Sistema Distribuído em Produção

**05/09/2025 (sexta-feira) - 08:00-12:00** 
- **Minicurso**: DevOps Desmistificado: Construindo sua Pipeline do Zero

## Objetivo
Apresentar aplicações práticas de DevOps e sistemas distribuídos para estudantes de computação, conectando teoria acadêmica com desafios reais do mercado.

## Conteúdo

### Palestra - Erlang, MCP e Kubernetes
Jornada pelos desafios reais de construir sistemas distribuídos para criar assistentes inteligentes com LLM. Explora integração BEAM/Erlang com MCP (Model Context Protocol), clusters escaláveis, consistência eventual via delta CRDTs e integração Kubernetes + BEAM.

### Minicurso - DevOps Desmistificado  
Guia prático desde fundamentos do terminal até pipelines CI/CD. Cobre Docker, automação de deploy e boas práticas DevOps usando aplicação Go como exemplo.

## Pré-requisitos (Minicurso)
- Linux ou WSL no Windows
- Docker instalado
- Editor de texto (VS Code recomendado)
- Conexão com internet
